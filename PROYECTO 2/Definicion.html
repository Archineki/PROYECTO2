¿Qué es un Navegador Web? (Definición)


Se conoce como navegador web (o simplemente navegador) o también explorador web (o simplemente explorador) a un programa informático que permite al usuario ingresar a las páginas Web que desee, siempre que conozca la dirección URL en donde se encuentra o bien que haga clic en un hipervínculo que conduzca a dicha página.



Un navegador web no hace otra cosa que conectarse a través de la Internet con el servidor en el que está la información que buscamos, y solicitarle las instrucciones de diseño y de ensamblaje visual que le permiten recuperar el texto, las imágenes y el ordenamiento de las mismas, para componer una página web y mostrárnosla ya terminada.

Pero, aunque todos los navegadores cumplen con esta misma función, no lo hacen siempre de la misma manera, y existen así navegadores más o menos veloces, dotados de ciertas características y compatibles con uno u otro Sistema Operativo (Etecé, 2021)
